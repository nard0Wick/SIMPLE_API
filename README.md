Once installed the necessary dependencies listed on the requirements.txt file, start the django server and follow the path: http://127.0.0.1:8000/clients/clients/

Before installing the listed dependencies it's higly recommended to create a python virtual environment:
$ python -m venv venv
For activating this virtual environment on linux:
$ source venv/bin/activate 
On windows:
$ venv/Scripts/activate

Once done proceed with the dependencies installation:
⬇️ pip install -r requirements.txt
And the server startup:
🚀 python manage.py runserver
